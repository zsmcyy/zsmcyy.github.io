Go1.18.3安装与配置

发表于 2022-07-10 更新于 2022-07-11 分类于 Go语言中文文档 本文字数： 677 阅读时长 ≈ 1 分钟

Go下载地址：https://golang.google.cn/dl/

Windows 安装 go 详细过程

下载 go1.18.3.windows-amd64.msi

下载过程

安装 go1.18.3.windows-amd64.msi，点击 Next

同意许可协议中的条款，点击 Next

选择安装位置，点击 Next

开始安装，点击 Install

安装过程

完成安装，点击 Finish

环境配置

打开开始菜单，输入 path 回车后打开 系统属性 对话框。点击 环境变量

在环境变量对话框中，将用户变量中的 GOPATH 删除，双击 Path，将 Path 中的 %USERPROFILE%\go\bin 同样删除。（这两项是 Go 安装完成后默认的配置，使用不方便，下面我们配置自己的环境变量。）

删除 GOPATH

删除 %USERPROFILE%\go\bin

点击 系统变量 下面的新建按钮，新建一个 GOROOT 变量，变量值为 Go 的安装路径。

继续新建一个 GOPATH 变量，变量值为自定义的 Go 工作目录（目录下需创建三个文件夹：src，bin，pkg）

找到系统变量中的 Path 双击打开，新建两项：%GOROOT%\bin 和 %GOPATH%\bin

自定义的 Go 工作目录

附加配置

解决国内不能下载插件的问题

系统变量中添加两项：分别是

  1. GO111MODULE，值为 on
  2. GOPROXY，值为 https://goproxy.cn,direct

检查

打开 bash，输入 go version，返回 go的安装版本

输入 go env，返回 go 的详细环境配置
